# Part3
 
This is my part 3 of the full stack open bootcamp

## Comandos

Install needed packages
```
npm i
````

Scripts to execute inside **package.json**

Most important ones: 

### Execute front
```
npm start
```

### Execute back
```
npm start
```

### Execute back *Dev*
```
npm run dev
```

### Deploy app on fly.io
```
npm deploy full
```
